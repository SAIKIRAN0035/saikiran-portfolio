Modern portfolio website with React, Tailwind CSS, and Framer Motion.

---

## Setup

```bash
git clone https://github.com/SAIKIRAN0035/saikiran-portfolio.git
cd saikiran-portfolio
npm install
```

Open `index.html` in a browser or use your preferred dev server.
